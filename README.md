CMSC 437 - Project 2: SkyLine Air In-Flight Infotainment System
================================================================

DESCRIPTION
-----------
SkyLine Air is a web-based in-flight infotainment system that allows
passengers to access entertainment, flight information, and in-flight
services during a flight. The application is built using HTML, CSS,
React 18 (via CDN), and Babel. No installation or dependencies are
required. Just unzip and open the HTML files in a browser using
Live Server or any local web server.
